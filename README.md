# Terminal Bitcoin Monitor

**!Work in progres!**

Command line monitor for the Bitcoin Network and your Bitcoin and Lightning node.

## Installation

`git clone https://github.com/jfrader/btcmon.git`

`cd btcmon`

`cargo install --path .`

## Usage

```sh
btcmon --bitcoin_core_host=localhost --bitcoin_core_rpc_port=8332 --bitcoin_core_zmq_hashblock_port=18332
```
